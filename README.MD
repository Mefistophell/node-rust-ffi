Node-Rust-FFI
=============

> Note: works only for Node.js v16.18

This is an example of using a dynamic library written in Rust within Node.js with FFI

## How to install and run

```
git clone https://github.com/Mefistophell/node-rust-ffi.git
cd node-rust-ffi
npm i
cargo build --release
node index.js
```
